This folder contains development-only demo data.

Rules:

- Never put demo data inside React components.
- Never modify repositories for demo purposes.
- Never modify database schema for demo purposes.
- Services decide whether to use demo data or Supabase.
- When DEMO_MODE = false, the application must use only live Supabase data.
